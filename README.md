## 💼 Проекты 

01. [Обзор метрик, оценка монетизации и юнит-экономики, поиск инсайтов и точек роста, A/B-тестирование](https://github.com/Natashe4qa/DA_projects/tree/main/01%20marketplace_unit_economics) 

***Описание:*** Комплексный анализ бизнес-показателей маркетплейса для выявления точек роста, оценки эффективность каналов привлечения, расчет юнит-экономики и предложение гипотез для улучшения продукта. Проведение А/В-тестирования и обработка его результатов.

***Стек:*** Python, Pandas, NumPy, Matplotlib, Seaborn, SciPy, Statsmodels.


02. [Изучение рынка заведений общественного питания Москвы](https://github.com/Natashe4qa/DA_projects/blob/main/README.md)

***Описание:*** Исследование рынка общественного питания Москвы для определения оптимальных параметров открытия нового заведения с целью максимизации его потенциальной успешности и прибыльности.

***Стек:*** Python, Pandas, Matplotlib.


03. [Ad-hoc запросы и дашборд для Агентства недвижимости](https://github.com/Natashe4qa/DA_projects/tree/main/03_real_estate_market)

***Описание:*** Агентству потребовался готовый анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости. Были решены несколько ad-hoc задач и предоставлены ответы на вопросы заказчика, а также подготовлен дашборд, отражающий состояние рынка жилой недвижимости в СПб и Ленобласти.

***Стек:*** PostgreSQL • Yandex DataLens
